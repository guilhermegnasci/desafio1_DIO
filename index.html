Estrutura Basica do HTML

(Voltar)

HTML / Head / Body

Pagina principal, inicial do site geralmente é denominada de “index.html”

Toda TAG no HTML ela tem sinal de menor, uma palavra e um sinal de maior.

HEAD (cabeça) - carrega antes de exibir para o usuário.

BODY (corpo) - visivel para o usuário.

Atributos das TAGs

(Voltar)

São propriedades que as TAGs podem ter.

São classificados de 2 maneiras: Atributos Globais e Atributos Específicos

Atributos Globais (Genéricos) - Todas as TAGs possuem;

Exemplo: id (identificador); style (css); class (classes);

Atributos Especificos

Exemplo: type (text/number - usado no input); src, (endereço/caminho da imagem - usado no img); width (largura)

Textos

(Voltar)

Tipografia HTML (buscar no Google);

Principais (mais usados):

  • h1 (maior), h2, h3, h4, h5, h6 (menor) - Usados em Títulos e Subtítulos;
  • p - Parágrafo;
  • blockquote - Citação. Da uma margem (da esquerda para direita);
  • Strong - Negrito;
  • u - Underline;
  • i - Itálico;
  • sup - Faz o texto ficar na parte de cima da linha;
